Título: Desmistificação do Git

Orador: Duarte Fonseca

Inscreve-te Aqui!

Nesta Game Talk o nosso orador irá dar algumas dicas sobre como utilizar o Git de forma eficaz no desenvolvimento de videojogos. O Git é uma ferramenta essencial para qualquer programador, especialmente na indústria dos videojogos, onde a colaboração e o version control são cruciais.

Qualquer pessoa está convidada a participar, e esperamos que esta sessão ajude a desmistificar o Git e a torná-lo uma ferramenta mais acessível e útil para todos os participantes.
